Aprendendo git

Estou aprendendo a utilizar o git.

Comandos básicos

git init - inicializa o git

git status - conferir se já tem arquivo upado ou não

git add "nome-do-arquivo.extensão" ou git add . - esse comando adiciona todos os arquivos que estão na pasta.

git commit -m "commit inicial" - confirações iniciais

git remotevadd origin "LINK" - Define para aonde vc quer fazer o upload do codigo

git push --set-upstream origin master - Envie os arquivos para o destino que você definiu anteriormente. Sempre que criar branch nova, utilizar este comando.

git push - Após utilizar o comando acima. Quando editar algum arquivo, basta utiliza este comando.

Comandos avançados

git reflog - Historico das versões com nome e id.

git reset --hard ID - Reseta para a versão que você colocar a ID.

git branch - Mostra as branch's disponiveis.

git branch NOME - Criar nova branch.

git checkout NOME - Seleciona branch que você colocar o nome.

git merge NOME - Vai juntar a branch que você selecionar. Lembrando que deve utilizar o comando na branch que você quer atualizar.
